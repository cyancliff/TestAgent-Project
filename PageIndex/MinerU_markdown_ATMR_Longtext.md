# 第3章 爱 TA 更要懂 TA

# 01 看懂左右脑：理性向左，感性向右

故事一：

在一个普通的小镇家庭里，住着小智和他的父母。小智的爸爸是一位严谨的工程师，他习惯于用逻辑和数据分析问题，而妈妈则是一位富有创造力的插画师，总是充满想象和艺术气息。

一天，小智垂头丧气地回到家，手里拿着一张满是错题的数学试卷，显然，这次的考试成绩并不理想。

小智的爸爸，一个典型的左脑使用者，接过试卷后眉头紧锁。他开始逐题分析小智的错误，用工程师的精确性指出每一个计算失误和逻辑漏洞。爸爸严肃地对小智说：“你看，这道题目的解题步骤应该是这样的…这里你忽略了一个重要的公式转换…”他一边说，一边用铅笔在草稿纸上列出详细的解题步骤，希望小智能够通过逻辑和条理来理解和掌握数学知识。

而小智的妈妈，作为一个右脑使用者，看到试卷后并没有马上纠结于错题，而是先给了小智一个温暖的拥抱。她轻声安慰道：“小智，别灰心，每个人都有失败的时候。也许你可以试着用另一种方式来理解这些题目，比如把它们想象成一场冒险游戏，数字就是你要收集的宝藏线索。”妈妈建议小智用画图的方式来理解复杂的数学问题，通过直观的图形和色彩来激发他的学习兴趣。

在这个例子中，我们可以看到爸爸更注重逻辑和条理，他倾向于通过分析和讲解来帮助小智理解数学知识；而妈妈则更注重情感和创造力，她试图通过想象和艺术的方式来激发小智的学习热情。

故事二：

心心是一个活泼好动的孩子，她热爱探索和创造。他的父亲是一位精确严谨的会计师，平时工作非常忙碌，注重事实和数据分析。而她的母亲则是一位热情洋溢的舞蹈老师，富有情感和艺术气息。

一个周末的下午，心心兴致勃勃地拿着他刚做的一个小发明一个简易的太阳能风车模型给父母展示。

父亲首先拿起风车模型，仔细观察其结构和运作原理。他开始提问心心关于风车的技术细节，如：“这个风车的叶片是如何设计的？太阳能板是如何转化能量的？”他试图通过逻辑和数据来理解这个发明。

在仔细分析后，父亲指出了一些设计上可能存在的问题，并建议心心如何改进以增加风车的效率。

父亲还帮助心心制定了一个改进和测试风车的计划，包括需要收集的数据和可能遇到的挑战。

母亲看到风车模型后，兴奋地抱起心心，赞美她的创造力和想象力。她对心心说：“这个风车真是太棒了！我能感受到你对这个项目的热情和努力。 ”

母亲建议心心可以尝试用不同的材料和颜色来装饰风车，使其不仅实用还更具观赏性。

母亲鼓励心心在学校的科学展览会上展示他的风车，并分享他的创作过程和心得。

在这个例子中，我们可以看到父亲更注重逻辑分析和事实依据，他通过提问和批判性思维来帮助心心改进风车的设计。母亲则更注重情感共鸣和创造力的培养，她通过赞美和鼓励来激发心心的创造力和自信心。

通过以上的两个故事，可以看出两个家庭的爸爸和妈妈是截然不同的教育方式，一个是理性和逻辑分析偏好，一个是感性、情感和创意的偏好，那么为什么会有父母教育模式的区别呢？这样的区别是不是有张可循呢？让我们一起来看一下非常著名的

“割裂脑”实验。

我们现在都比较熟知的“左右脑”理论，来源于美国教授罗杰斯佩里“割裂脑”实验。通过割裂脑实验，证实了大脑不对称性的“左右脑分工理论”。他发现，左脑主要负责逻辑理解、记忆、时间、语言、判断、排列、分类、逻辑、分析、书写、推理、抑制等功能，而右脑则主要负责空间形象记忆、直觉、情感、身体协调、视知觉、美术、音乐节奏、想像、灵感等。由于罗杰·斯佩里在割裂脑实验和左右脑功能分工方面的杰出贡献，他于1981 年荣获诺贝尔生理学或医学奖。这一奖项是对他在神经科学研究领域所取得成就的极高认可。

# 02 揭晓性格特质：ATMR 慧识人测评

在左右脑理论中，人们对于左右脑使用偏好的不同，会影响决策偏好，有的决策偏情感，关注人，有的决策偏逻辑，关注事，我们还发现了人们因为信息获取偏好的不同，存在着做事快与慢的区别，对于和正常人相比，做事快的，我们称之为直觉型信息获取偏好，对于和正常人相比，做事慢的，我们称之为分析型信息获取偏好，

所以，从情感到逻辑，从直觉到分析，结合中国人自身的文化特点，ATMR 把人的特质划分为四个象限，A（Appreciation）是欣赏型，关注人，做事快，比较感性；T（Target）是目标型，关注事、做事快、理性，M（Magnanimity）是包容型，关注人，做

事慢，感性；R（Responsibility）是责任型，关注事，做事慢，理性。

ATMR 慧眼识娃测评系统是结合青少年儿童心理学、行为学、脑科学等各科学理论基础及大数据分析，研发出的专门针对中国3-14 岁青少年儿童性格特质的测评工具。

ATMR慧识人测评系统是结合管理学、心理学、脑科学、行为学、中国传统文化等各学科理论基础及大数据分析，研发出的专门针对14 岁（含）以上青少年及成人的性格特质测评工具。

测评中得分最高的特质为主要特质，这个特质决定一个人超过 $60 \%$ 的思维、情绪和行为的风格。得分第二的特质为次要特质，依次类推为第三和第四特质，正是这些个性特质的主次地位以及强弱表现的不同排列组合，造就了每个人成为独一无二的个体。排列组合数目不胜枚举，但是根据共性的主要特质和次要特质组合，我们本书中主要展示了主要特质和次要特质联动的大12 类不同特质的模式：

直觉均衡模式(AT、TA）:

左右脑均衡型，普遍做事比较的快，直觉性非常的好，能同时关注到人和事

分析均衡模式(MR、RM）：

左右脑均衡型，普遍做事比较的慢，做事相对比较稳健，能同时关注到人和事

单一左脑模式(TR、RT）：

单一理性脑区，做事理性，关注逻辑，比较在意事情

单一右脑模式(AM、MA)：

单一感性脑区，做事感性，关注情感互动，比较在意感受

对冲模式：AR 、MT 、TM 、RA，冲突型特质，平衡的好，是个能力阈值非常大的孩子，平衡的不好，就是个容易自我消耗型的孩子。

# 03 天赋优势

美国教育学家、心理学家霍华德·加德纳说：“每个孩子都是一个潜在的天才儿童，只是经常表现为不同的形式。”作为家长，我们的责任就是去发现并珍视孩子的这些天赋优势，为他们提供适宜的生长环境和营养土壤。

那么，什么是天赋优势呢？简而言之，天赋优势是指孩子与生俱来的、在某些方面超越常人的能力和特质。这些能力和特质可能是智力上的，如逻辑思维、空间想象力；也可能是情感上的，如同理心、共情力；还可能是身体上的，如运动能力、手眼协调等。

天赋优势，越早发现越好，天赋优势是孩子成长的独特密码，是他们内心深处的声音和力量。当我们了解并尊重孩子的天赋优势时，就能激发他们的学习热情，培养他们的自信心和创造力。相反，如果我们忽视或压抑孩子的天赋优势，就可能会阻碍他们的成长和发展，甚至导致他们产生自卑和逆反心理。

因此，了解孩子的天赋优势，是每位家长必须承担的责任。我们要用心去观察、去倾听、去理解孩子，发现他们的独特之处，并给予积极的反馈和支持。只有这样，我们才能帮助孩子解锁他们的天赋密码，让他们在成长的道路上更加自信、坚定和快乐。

在这个多元化的时代，每个孩子都有机会展现自己的才华和魅力。

现在，让我们一起走进两位名人的童年，感受他们天赋优势的璀璨光芒。一个小男孩，从小他就对数学和物理充满了浓厚的兴趣，常常陷入深深的思考，眼中闪烁着对宇宙的好奇，他的家人敏锐地捕捉到了这一点，鼓励他探索科学的奥秘。正是这种对科学的执着和天赋，让他最终揭开了相对论的神秘面纱，成为科学界的巨星，他就是爱因斯坦。

再来看另一个故事，一个小男孩坐在钢琴前，双手在琴键上跳跃，美妙的旋律流淌而出，他就是未来的音乐大师贝多芬。从小，他就对音乐有着异乎寻常的感悟力，能够轻易地捕捉到音乐中的情感与节奏。他的家人被他的才华所震撼，全力支持他的音乐之路。正是这种与生俱来的音乐天赋，让他在音乐的殿堂里留下了永恒的印记。

了解孩子的天赋优势，如同掌握了一把神奇的钥匙，能够打开孩子内心深处的世界。当我们发现并尊重孩子的天赋优势时，就能激发他们的学习热情，培养他们的自信心和创造力。因此，作为家长，我们必须承担起了解孩子天赋优势的责任，用心去观察、去倾听、去理解孩子内心的声音和力量。找到属于孩子的真正的内在天赋，家长才能实现“因材施教”。因为对孩子来说，做的事情，如果与其天赋优势相吻合，他会充满热情，越来越自信，进步的速度会远超其他人。

孩子成长的过程中确实面临着许多不确定性，在成长过程中，家长作为孩子的重要引导者和支持者，我们的决策和行动对孩子的成长具有深远的影响。在接下来的篇章中，我们也会着力就每个特质的天赋优势，给出针对性的建议。

是他们适应环境的一种方式。

# TA特质孩子的心理探秘

TA 特质的孩子，他们通常具有一种独特的思考方式，那就是倾向于直觉思考。这种思考方式让他们能够迅速地把握事物的本质，看到问题的深层含义，他们的喜怒哀乐也相对较为明显，都毫不掩饰地展现在脸上。他们往往对于自我期许很高，对待目标和期望都会尽力追求。因此，当他们发现自己未能达成既定的目标时，内心的压力会渐渐积累。这种压力可能会表现为急躁和情绪化，他们可能会显得不耐烦，容易发火，或是情绪低落，沉默寡言。

然而，尽管他们可能在外表上显得情绪化，但实际上，他们内心是渴望理解和沟通的。他们喜欢表达自己的想法和感受，愿意与他人分享自己的内心世界。这种开放和坦诚的态度，使得他们在面对压力时，能够找到有效的途径来释放和缓解。此外，他们通常也具备非常不错的抗压能力。

TA 特质的孩子可能表现出较为明显的情绪反应和对目标的执着追求，但他们的开放性和抗压能力使得他们在面对压力时，能够通过表达和沟通来减轻内心的负担，从而避免心理问题的产生。

他们的坚韧和乐观，使得他们在成长的道路上，能够勇往直前，不断超越自我。

TA 特质的孩子，其心理压力主要源自未能实现自我设定的目标，以及感到自己的声音未被他人倾听和理解。由于这类孩子可能缺乏一定的同理心，他们可能展现出更为强势和挑衅的行为，容易与他人发生冲突。在探索这些孩子的心理问题时，我们应当首先关注并释放他们的情绪。了解他们未能达成的目标，以及这为何会让他们感到沮丧，通过提问和倾听，他们可能会主动分享内心的想法和感受。给予他们空间去表达情感，如哭泣或发泄，有助于减轻他们的情绪压力。当他们释放了内心的负担后，他们可能会自然而然地恢复平静，并找到解决问题的途径。

总的来说，TA 特质的孩子通常具备独立的判断能力和明确的目标感，这使得他们在追求成果的过程中可能会面临一定的压力。当未能达到预期目标时，他们可能会表现出急躁和情绪化的特点，甚至显得较为强势和不近人情。然而，值得注意的是，这类孩子通常能够直接表达自己的情感和需求，不太倾向于将问题积压在心里，从而降低了出现心理问题的风险。

# 02 单一左脑模式(RT)

# RT 特质孩子的天赋优势

R和T 都是属于逻辑型特质，所以比较理性，逻辑性好，有目标感，对于认定的目标能够坚持，在实现目标的路上非常坚韧、有耐力、能够接受挑战，自己想要的事物，想尽办法去完成和达到,执行力很强，凡事有交代，说到做到。

RT 特质的孩子，逻辑缜密，善于分析，思考问题的时候，有自己的想法和见解，比较讲道理，善于分析总结找方法，会迅速找到问题的客观原因。非常坚韧有耐力，爱分析爱钻研，通常都会全面、系统、客观的去看待问题。一般学理科会非常有优势，数学、物理、化学等等，孩子都能很容易取得不错的成绩。

RT 特质的孩子，抗挫折能力很强，做事情很理性，先确定目标，然后以目标为导向，经过缜密细致的分析，列出做事计划，条理清晰，扛得住压力，喜欢不断获胜的结果。即使遇到了困难，他会及时调整目标，调整方法，除非差距过大，一般而言，RT 特质的孩子，不会轻易放弃自己设定的目标，也愿意去坚持不懈，所以在学习上，特别容易拔尖，特别容易成为小学霸。

RT 特质的孩子，探索欲很强，喜欢去探索，喜欢研究事物的本质，喜欢找原因，一件事情一定会刨根问底，思考比较多，比较深刻，常常让家长觉得孩子比较早熟。

RT 特质的孩子，非常有责任感。答应的事情，一定会完成承诺，这类特质的孩子比较省心，只要是他自己认可的事情，想做的事情，他会非常自律的去完成，不太需要家长过多的关注或者催促；很注重自己的承诺，除非不答应，一旦答应，一定会完成。

RT 特质的孩子，很有主见，独立自主，一旦制定了计划，确定了目标，是自己想做的，会马上行动，全力以赴，即使有困难，会根据自己的节奏完成，有自己的主意，而且希望家长都听自己的，也不会轻易放弃目标。这种特质的孩子，不惧怕比赛，他渴望获得胜利，所以适当的压力，反而会激发起他的野心，他一旦

想做一件事情，会竭尽全力，遭遇失败的打击，也不会特别气馁。非常有自信，有自己的独立人格，认为自己只要认真去做，一定可以完成，行动力比较强。

RT 特质的孩子，目标感清晰，对于认定的目标能够坚持，在实现目标的路上非常坚韧、有耐力、且愿意接受挑战；对于目标有自己的计划，路径清晰，即便暂时遇到了困难，仍能在家长或老师给出合理化建议和批评后，继续坚持，一次次的尝试，甚至遭遇失败的打击也不气馁，自己想要的事物，想尽办法去完成和达到。

RT 特质的孩子，专注力很好，当他投入做一件事情的时候，非常专注，可以沉浸在自己的世界中，意志集中，仿佛全世界的喧嚣都被隔绝在外，完全不受外界的干扰。

# RT 特质孩子的潜在弱势

RT 特质的孩子，因为想的比较多，所以做事情不会立刻就行动，做事情有点完美主义倾向，他们不喜欢犯错，所以会反复思考，希望能够分析出最好的方法再去做，所以容易在一件事情上耗费太久的时间。学习上，家长如果不明白这个道理，会认为孩子怎么这么磨蹭拖拉，但其实不是，他们只是思考的太多。举个例子，我有个朋友家孩子，典型的 RT 特质，做事情非常讲究完美。有一次我们出去吃饭，饭店给他的儿童餐盘，由于用了很久，餐盘中间有一个小黑点，怎么擦也擦不掉，他就不接受，一直闹，非要换一个才能安心吃饭。这就是完美主义倾向。

RT 特质的孩子，做事情有的时候太固执，一根筋，太相信自己的思考，所以会一直坚持自己的想法，不太容易接受别人的建议。因为你的孩子喜欢稳定，不喜欢变化，所以相对应，做事情的时候，不太容易变通，容易钻到牛角尖去。

RT 特质的孩子做事比较慢，尤其是不熟悉的领域，面对新事物的时候，或者自己没有把握的前提下，对完成事情非常想做到完美，想做到面面俱到，心里想太多，思考得太久，会在脑海中不断推演，所以迟迟不能行动，有时候因为想太多会错过很多机会。

RT 特质的孩子，由于有完美主义倾向，所以对自己和对别人的要求都比较高。当别人做错事或者没做好的时候，他不太会在乎别人的感受，会直来直去的批评，有一些苛责。

所以我们说，RT 特质的孩子，同理心没有那么强，在乎事情远远超过在乎人；做事的时候，更多是按照自己的目标和计划来，不会去太过于包容别人的想法。更多关注目标的顺利完成，对于别人的感受不是很在乎，所以往往会有一部分人会觉得他很强势，不是很喜欢与他交朋友。在人际关系中，他的朋友没有那么多，但是都比较铁的。

RT 特质的孩子，控制欲有点强；对自己和别人的要求都很高；在与人沟通方面，会有些缺乏弹性。

# RT 特质孩子的培养方向

RT 特质的孩子拥有仔细钻研的研究精神，一旦下定决心做某件事情，都可以完成很好的结果。他具有深入探索思考的潜能；所以从小可以多带孩子玩一些探索、发明类的小游戏，孩子会非常来劲，也能很好地锻炼他的创造力。关于兴趣班方面，类似科普、手工等方面，能很好的释放孩子的想象力；类似模型搭建、建构积木、小小发明等兴趣班，可以很好的让RT 特质的孩子释放动手潜能，激活思考实践能力，有很强的发明创造天赋。

RT 特质的孩子，一般比较稳重，很坚韧，不是那么爱动，但是有目标感。所以，我们一般建议，RT 特质的孩子，可以适当的往运动类方向去报兴趣班，从锻炼身体和发挥孩子逻辑分析特长的角度，比如说乒乓球、羽毛球、跆拳道、击剑、冰壶等等等等；报这些类型兴趣班，

RT 特质的孩子，目标感强，逻辑思维清晰，计划性强，善于分析找方法，抗压能力高；独处时偏向于安静，不管做什么事情，只要自己认可了，有目标有方法，稳步推进。所以，如果是发挥孩子的优势，在逻辑和分析占比高的领域可以多涉及，比如编程、围棋、奥数等，能进一步提高孩子的逻辑思维、统筹规划能力。你的孩子在学这些兴趣班的时候，既能够坐得住，同时又能进行深入的思考，所以通常比其他个性特质的孩子，进度速度要快很多，如果恰好又有天分，是值得深入去培养的。

RT 孩子并不擅长临时的发言演讲，但只要给予充足的时间，做好准备，不是“突如其来”、“临时起意”，让孩子对可能发生的后果做好预案，并多多练习，那么一个自信大方，能量满满的RT 宝宝绝对会惊艳全场。RT 特这的孩子期望自己做的事情是经过深思熟虑的，所以自己的发言、或者文字，能够按照预期影响到别人的时候，他会感觉到被欣赏，会充满自信，家长可以用不断的用高光时刻，来增强他的能力和信心。

题多发特质。

# RA 特质孩子的心理探秘

教育家陶行知先生在针对教育有一句至理名言：培养教育人和种花木一样，首先要认识花木的特点，区别不同情况给以施肥

浇水和培养教育，这叫“因材施教”。孩子就像这些花木一样，认清特点，才能因材施教。

RA 特质的孩子，同时包含了逻辑性的分析和直觉化的情感特质，所以在外在表现上，偶尔会有内在冲突性。大部分时候，孩子比较沉稳冷静，刚接触一个陌生的环境时，他/她不会一下子就融入，会不动声色地先观察，会有自己的判断，但是一旦自己认可了，顺利融入以后，又会成为团队中的氛围小能手，会活泼好动、有表现欲，希望可以成为焦点。

RA 特质的孩子对新事物充满好奇，喜欢探究十万个为什么，喜欢思考，喜欢稳定的计划安排，喜欢被表扬，不喜欢被批评，抗压能力比较强。孩子的变通能力不够，有的时候思考会钻牛角尖，如果想不通就会有情绪化，一旦情绪没有得到很好的释放，很容易大起大落，一根筋走到底，谁的话也不听，家长需要给孩子一些自我思考的时间和空间。

对于RA 特质的孩子来说，做这件事对不对，比这件事能不能做好更重要。所以我们在梳理孩子心理问题的时候，也要秉承这个优先级，要注重理解孩子对于事情意义感的理解，就是要知道他自己是怎么认为这件事情的。因为你的孩子有自己的一套道理，一旦固执起来的时候，会死活不听你的道理，这个时候家长一定要稳住，千万不要只是批评，而是你要让他把心里话都说出来，然后你要肯定他的判断，提出一些问题和具体的事例来给孩子启发，让他自己找到正确的道理，自己去思考对不对。少讲大道理，多去引导他自己想，只有他自己觉得自己错了，或者要怎么改，他才能心态平和的解决问题。
