## 🎯 智能选题算法原理

### 能力向量计算
```
能力向量 = Σ(题目特征向量 × 得分权重)
权重 = 归一化得分 × (异常作答 ? 0.5 : 1.0)
```

### 题目评分公式
```
分数 = 0.4 × 特征匹配度 + 0.3 × 难度匹配度 + 0.3 × 区分度
特征匹配度 = |cosine_similarity(能力向量, 题目特征向量)|
难度匹配度 = 1 - |题目难度 - 估计能力|
区分度分数 = 题目区分度
估计能力 = 特征匹配度 (简化模型)
```

### 首次选题策略
```
分数 = 0.6 × 难度匹配度 + 0.4 × 区分度
难度匹配度 = 1 - |题目难度 - 0.5| × 2 (越接近0.5分数越高)
```
