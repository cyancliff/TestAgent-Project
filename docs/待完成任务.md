## 数据库迁移命令备忘

```bash
# 安装 Alembic
pip install alembic

# 初始化
alembic init alembic

# 创建迁移脚本
alembic revision --autogenerate -m "分离报告表"

# 执行迁移
alembic upgrade head

# 回滚
alembic downgrade -1
```
