# 多模态数据忽略说明

## 目的

多模态人格模块会产生大量原始数据和中间文件，包括：

1. 原始视频压缩包
2. 解压后的视频文件
3. 标注与转写 pickle
4. 预处理工件
5. 特征文件
6. checkpoint

这些内容不应该进入版本库，也不应该被代理重复扫描。
